# Data Analysis Projects

This repository contains a curated collection of **data analysis projects** demonstrating skills in **Excel, SQL, Python, and Power BI**. Each project showcases real-world data cleaning, transformation, visualization, and exploratory analysis, providing end-to-end insights for business decisions.

## Project Categories

### Excel Projects
Projects focus on **data cleaning, pivot tables, pivot charts, and advanced Excel features** to analyze and summarize business data.

### SQL Projects
Projects involve **data cleaning, transformations, aggregations, and exploratory data analysis** using SQL. Window functions, CTEs, and ranking methods are applied to extract actionable insights from structured datasets.

### Python + SQL + Power BI Projects
End-to-end projects using **Python for data preprocessing, SQL for querying and transformation, and Power BI for visualization**. These demonstrate full workflow capabilities from raw data to interactive dashboards.

## Key Skills Demonstrated
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- SQL queries, aggregations, and window functions
- Python for data manipulation
- Interactive dashboards in Power BI
- Business insights and recommendations

## How to Explore
- Navigate through the folders to access individual projects
- Each project contains a **README** describing the dataset, cleaning steps, analysis, and key takeaways
- SQL scripts, Excel files, Python notebooks, and Power BI files are included where applicable
