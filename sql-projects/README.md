# SQL Projects

This folder contains SQL-based data analysis projects focused on **data cleaning, transformation, and exploratory data analysis (EDA)** using real-world datasets.

The projects demonstrate practical SQL skills such as:
- Data cleaning and standardization
- Use of CTEs and window functions
- Aggregations and ranking analysis
- Trend and time-series analysis

All projects are designed to reflect real business use cases and are suitable for portfolio and interview discussions.
